# Installation

## Stable release

To install mlx-embeddings, run this command in your terminal:

```
pip install mlx-embeddings
```

This is the preferred method to install mlx-embeddings, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install mlx-embeddings from sources, run this command in your terminal:

```
pip install git+https://github.com/Blaizzy/mlx-embeddings
```
